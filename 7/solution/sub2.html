E-Contract

온라인으로 완결되는 전자계약

전자계약은 인터넷으로 계약서를 주고받고, 공인인증서로 서명하여 계약을 체결하는 방식입니다. 종이 계약서의 인쇄, 우편, 보관 과정을 모두 생략하고 온라인에서 간편하게 계약을 마무리할 수 있습니다.

ERP 시스템과 연동되어 이미 입력된 정보를 불러와 계약서를 자동으로 생성하기 때문에 2중 입력이 필요 없으며, 여러 문서를 하나로 묶어 통째로 서명하는 방식으로 간인 절차도 사라집니다.

전자계약 서비스 이미지경로

Expected Benefits

기대 효과

2중 입력 방지

ERP와 연동되어 이미 입력된 정보를 불러와 계약서를 자동 생성합니다. 같은 데이터를 두 번 입력하는 번거로움이 사라집니다.

간인 필요 없음

여러 문서를 하나로 묶어(Zip) 통째로 서명하기 때문에 문서 누락이나 변조가 불가능하며, 종이 계약서의 간인 절차가 필요 없습니다.

인지세 절감

종이값, 우편비는 물론이고 전자계약 시 인지세도 절감됩니다. 계약 건수가 많을수록 절감 효과가 커집니다.

가점 혜택

건설업자 간 상호협력 평가나 PQ 심사 때 신인도 가점을 받을 수 있어 영업에도 유리합니다.

Support Tool

CoreContractManager

CoreContractManager

계약서 관리를 자동화하는 전용 툴

사용자가 계약서를 일일이 다운로드하고 업로드하는 과정을 자동화한 전용 프로그램입니다.

자동 압축/해제

계약서, 내역서, 특수조건 등 수많은 파일을 하나로 묶어 관리합니다.

실시간 변조 검사

로컬 파일과 서버 파일을 비교해 0.1mm의 수정도 즉시 감지합니다.

원클릭 조회

프로그램 안에서 더블 클릭하면 Excel, Word 등이 바로 실행됩니다.

자동 업로드

서명만 하면 서버로 자동 전송됩니다. 파일 찾기 버튼을 누를 일이 없습니다.
